# MMO LFG

Платформа для поиска игроков в MMO-игры. Пользователи создают объявления, другие игроки откликаются.

## Функционал
- Создание и редактирование объявлений
- Отклики на объявления со статусом
- RichText-редактор для постов
- Авторизация через Django

## Стек
- Django
- SQLite
- CKEditor

## Запуск

```bash
python manage.py migrate
python manage.py runserver
